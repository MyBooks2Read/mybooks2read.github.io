213 7.1 Going beyond the interpolation from sympy import * from math import pi x, a, b, c, d = symbols('x a b c d') sine = a*x**7 + b*x**5 + c*x**3 + d*x sine_d = diff(sine, x) sine_i = integrate(sine, x) the_system = [ sine.subs(x, pi / 2) - 1, sine_d.subs(x, 0) - 1, sine_d.subs(x, pi / 2), sine_i.subs(x, pi / 2) - sine_i.subs(x, 0) – 1 ] res = solve(the_system, (a, b, c, d)) Now, this model is fully comparable to the one we got from the series in terms of com- plexity. It has four coefficients, like the one from the series. And the coefficients apply only to even degrees, like with the one from the series. The models are almost exactly alike, apart from some numeric differences in the coefficients (see table 7.1). The new model, however, is more precise than the power series. Its mean error is 0.000001, which is more than ten times smaller than the 0.000018 from before.
