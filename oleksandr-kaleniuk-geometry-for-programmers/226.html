202 CHAPTER 6 Polynomial approximation and interpolation  You can approximate a set of data points by using the least-squares method.  Interpolation is the special case of approximation in which the interpolating function goes strictly through all the interpolating points.  Normally, a set of N data points is interpolated with (N –1)-degree polynomial, but using higher degrees is also possible.  You can use the least-squares method to interpolate a data set, but using the Vandermonde approach is more practical computationwise.  Approximation (and interpolation in particular) helps us build nice graphs, but their applicability for real-world modeling is a matter of data science, not geometry.
