All the algorithms in the C++ standard library, for example, have their complex- ity specified in big-O. So when we pick an algorithm for a job, we can look briefly at what the standard says about its complexity and make a well-reasoned choice. 3.4.3 Section 3.4 summary Some algorithms for solving linear equation systems aren’t iterative, but direct. They operate on different principles that allow them to solve systems in a predictable amount of time. Although the efficiency of iterative algorithms is characterized mostly by their con- vergence, direct algorithms should be assessed by using complexity estimation instead. This complexity estimation usually comes in big-O notation, which is a func- tion showing how the amount of abstract operations grows with the dimensionality of the problem. 3.5 Linear equations system as matrix multiplication We’ve done something similar before.
