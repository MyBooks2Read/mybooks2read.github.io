To do so, we’ll compute tangent vectors in each segment’s vertices and substi- tute a cubic polynomial curve for each segment. We’ll also split the curves into 16 little linear segments each so we can draw them easily, but that’s a technicality. Repre- senting curves as approximated sets of segments makes plotting simpler, but it isn’t essential to the algorithm.
