Interpo- lation can help you with only the former task. Numeric computations in Python So far, we’ve used SymPy to do all our math. But SymPy is meant to deal with sym- bolic computations—hence the Sym in the name. In our examples, we compute poly- nomial coefficients as numbers and don’t care which symbols they’re assigned to. So normally, we would have chosen NumPy for the job instead. The Num part of the name means “numeric,” and it’s a library for efficient numeric computations. Its web- site (https://numpy.org) has extensive documentation. As an example, this is how you declare a matrix in NumPy: a = np.array([[1, 2], [3, 5]]) This is how you declare a column vector: b = np.array([1, 2]) And this is how you solve a system of linear equations: x = np.linalg.solve(a, b) Yes, it’s that simple.
