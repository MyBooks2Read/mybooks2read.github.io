The final result, with cubic curve pieces instead of line seg- ments, looks like figure 11.27. 11.4.2 The implementation In Python, solving the cubic equations looks like this: from sympy import * x0, y0, x1, y1, dx0, dy0, dx1, dy1 = symbols('x0 y0 x1 y1 dx0 dy0 dx1 dy1') ax, bx, cx, dx, ay, by, cy, dy = symbols('ax bx cx dx ay by cy dy') Figure 11.27 The final result is built with the segments from before turned into cubics by adding the tangent vector equations. Input data: Points (x0, y0), (x1, y1) and the tangent vectors (dx0, dy0), and (dx1, dy1) Output: The coefficients of the two polynomials Px(t) = ax*t**3 + bx*t**2 + cx*t + dx, Py(t) = ay*t**3 + by*t**2 + cy*t + dy that describe the target parametric cubic.
