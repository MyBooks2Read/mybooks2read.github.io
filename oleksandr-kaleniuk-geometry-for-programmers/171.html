Which one should we pick? The problem isn’t really mathematical; both solutions will give us valid biarcs. So let’s pick an arbitrary rule that seems pragmatic. How about computing both solutions numerically and picking the solution with the smallest sum of radii? That should work, and it kind of does. We have our biarc, and it’s nice and smooth. The only problem is that it’s a little “bubbly” when the tangent vectors point in roughly the same direction (figure 5.20).
