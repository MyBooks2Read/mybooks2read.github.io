It works for a single point resulting in a constant function f (x ) = y 1 . For a pair of points, it results in a straight line. For three points, it makes a perfectly fitting quadratic polynomial (fig- ure 6.14). For all the points more than three, it still results in a quadratic polynomial, but we compute only three coefficients so what else could it be? Due to the rather elaborate math, which we won’t get into, the least-squares error metric is guaranteed to be minimal for this polynomial on any set of points, no matter how large the set is (fig- ure 6.15).
