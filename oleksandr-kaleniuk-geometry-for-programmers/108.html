Exercise 3.5 Here’s something to challenge your hyperspatial intuition. Suppose that a ray in 4D is given as a point vector to its origin P and a vector of its direction d. Also suppose that a 3D singular cube is in this space. Because it’s 4D space and only a 3D singular cube, it has to be flat in one of the dimensions, so let’s say it’s the fourth. Then cube C in parametric form is C = (1, 0, 0, 0) i + (0, 1, 0, 0) j + (0, 0, 1, 0) k, where i, j and k are in range [0, 1]. Note that there’s no (0, 0, 0, 1). The last coordinate of this cube is always 0.
