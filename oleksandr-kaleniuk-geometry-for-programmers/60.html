But the sine function also has a distinct range, which is not all the numbers, but specifically [-1, 1]. The range is a separate term, not the same as the codomain. Speaking in pro- gramming terms, the codomain is the output type of the function, whereas the range is all the possible values the function can produce. The sin(x) and tan(x) share the codomain , but tan(x) covers all the with its range, and sin(x) does not (figure 2.21). The domain, confusingly, isn’t the type, as in a codomain, but all the possible values, as in a range. This time, it’s all the possible values for the input. Here’s another important difference between the sine and the tangent: the former is continuous, and the latter is not.
