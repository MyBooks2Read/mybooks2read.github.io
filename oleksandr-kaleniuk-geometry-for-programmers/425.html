The full source code, with all the utility functions and plotting, is available at ch_12/ image_vectorization.py in the book’s source code. 12.4.6 Section 12.4 summary The image vectorization algorithm we implemented may not be practical. Yes, we did turn a raster image of an arrow into a set of curves, and we did keep the formal criteria for smoothness and continuity. But an artist with a vector graphics editor will draw a much better arrow in a minute. For any practical use, we need to employ more elabo- rate algorithms.
