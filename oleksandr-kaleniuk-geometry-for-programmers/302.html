Have fun! 8.5 Solutions to exercises Exercise 8.1 A single-variable cubic interpolation requires four points. Quadratic requires three. This multiplies into the 12-point requirement for an unambiguous cubic-quadratic interpolation. Exercise 8.2 If we allow ambiguity, we can go with fewer points. The same logic applies here as with a single-variable interpolation. We can run a parabola through a pair of points; in fact, we can run an infinite amount of parabolas through a pair of points. But we can’t make a straight line run through three points unless they lie on the same line. And because we expect our points to be random, we have to assume that they don’t. So “At most 12” is the correct answer. Exercise 8.3 We can make a surface that looks like a cylinder, but not a precise model of a cylinder. That’s what we need NURBS for, remember? With a rational spline, we can model a cylindrical surface, but with polynomials, we can only approximate the cylinder within some tolerable error.
