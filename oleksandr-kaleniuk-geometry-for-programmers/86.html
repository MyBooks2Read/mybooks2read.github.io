The factorial function is a common interview question for junior developers. It’s funny that the most effective implementation for lim- ited integers in C++ or Java is not a real function, but a short lookup table. So the direct algorithm, although it always works in N iterations, becomes complex when N grows large. At some point, it becomes impractically complex. This brings us to the question of algorithm complexity. How do we assess that? 3.4.2 Algorithm complexity We usually don’t estimate algorithm complexity by counting actual processor cycles.
