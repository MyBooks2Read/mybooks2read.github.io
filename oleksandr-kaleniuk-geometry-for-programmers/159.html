Call it on everything you want: >>> diff(x**3 + 2*x**2 + sin(x)) 3*x**2 + 4*x + cos(x) A function may have several symbols in it. Some symbols may be variables, and some may be coefficients. SymPy doesn’t know which is which, so you may want to help by stating the variable explicitly as the second parameter of the diff: >>> diff(a*x**2 + b*x + c, x) 2*a*x + b Some functions are multivariable. A parametric surface, for example, is a function that connects pairs of coefficients and points of a 3D space. A single component of this connection is a function of two coefficients. You can differentiate this function by one or even both in one go by giving SymPy explicit instructions: >>> diff(a*u**2 + b*v**2*u + c, u) 2*a*u + b*v**2 >>> diff(a*u**2 + b*v**2*u + c, v) 2*b*u*v >>> diff(a*u**2 + b*v**2*u + c, u, v) 2*b*v We have to set x explicitly. From SymPy’s perspective, a*x**2 + b*x + c is a generic symbolic expression, not necessarily a function of x. Both u and v are variables that we can use for differentiation.
