There’s some error, but it’s less than the naked eye can notice, and because we’re talking about data visualization, the plot kind of works (figure 6.35). 6.4.3 Section 6.4 summary You can use polynomial approximation and interpolation to show data obtained as a set of points. When you want to show the general trend, you can use approximation. When you want to show data between data points, use interpolation. With the power of engineering, you can make them both work together. On each iteration, exactly one new last point is added. But you can add more if you want the algorithm to converge faster.
