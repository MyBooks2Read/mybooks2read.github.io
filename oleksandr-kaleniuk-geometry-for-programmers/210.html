But let’s not give up on simple polynomials yet. 6.3.3 How to lessen unwanted oscillations In general, we have no control over waving. All we have is the data set, and the data set governs how the interpolating function goes. But what if we’re allowed to choose how to compose this set? What if we have the xs set for us beforehand and are allowed to choose only ys to set the function? As it turns out, we have a way to place xs on the x-axis to minimize the oscillations.
